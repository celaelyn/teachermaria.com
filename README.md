# Teacher Maria Nascimento - Site Oficial

Esta pasta contem a versao oficial do site para publicar no Vercel.

## Arquivos principais

- `index.html`: site principal, com apresentacao das aulas e demonstracao visual do portal.
- `portal.html`: pagina separada da area do aluno, com login e portal do aluno.

## Como publicar no Vercel Drop

1. Compacte ou envie esta pasta inteira: `teacher-maria-oficial-publicacao`.
2. No Vercel, use a opcao de upload/drop.
3. Depois que o deploy terminar, confira o link principal do projeto.

## Como editar depois

O arquivo principal do site e `index.html`.

Se o projeto for conectado ao GitHub, qualquer alteracao enviada para o repositorio pode atualizar o site automaticamente no Vercel.

## Observacao importante

Esta versao deixa a presenca online pronta, separa o site principal do portal do aluno e apresenta um acesso inicial de teste. Ela ainda nao cria login real com banco de dados sozinha.

Para login privado, pagamentos automaticos, envio real de lembretes, contratos individuais, atividades salvas e correcao com IA funcionando com dados reais, o proximo passo e conectar:

- autenticacao de alunos;
- banco de dados;
- painel da professora;
- servico de pagamento;
- servico de e-mail/WhatsApp;
- correcao com IA.
